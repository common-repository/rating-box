== Installation ==
#### Thank you for your interest in Rating Box.
### Minimum requirements.  
*   Wordpress 3.0+  
*   PHP 5.x  
*   MySQL 5.x

This section describes how to install the plugin and get it working.
e.g.


1. Upload folder 'plugin-name' to the `/wp-content/plugins/` directory.
2. Activate the "Rating Box" plugin through the '[Plugins](http://photoboxone.com/category/plugins/)' menu in WordPress.
3. Go to settings.

If any problem occurs, please read [documents wordpress](http://photoboxone.com/category/documents/).
